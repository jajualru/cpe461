Generic Structure:
                      8-bit                   32-bit                           32-bit
                 ---------------- -------------------------------- --------------------------------
                | Op code        | Value 1                        | Value 2                        |
                 ---------------- -------------------------------- --------------------------------


RasPi -> Arduino Commands:
pwm_write         0b00000001       pin#                             value
digital_write     0b00000010       pin#                             value
encoder_read      0b00000100       pin#                             (unused)
analog_read       0b00001000       pin#                             (unused)
digital_read      0b00010000       pin#                             (unused)
distance_start    0b00100000       (unused)                         (unused)
distance_poll     0b00100001       (unused)                         (unused)
serial_write      0b01000000       str_length                       char1...charX (extend packet as necessary)
LCD_write         0b10000000       TODO                             TODO
LCD_clear         0b10000001 (129) (unused)                         (unused)
pin_init          0b11110000 (240) pin#                             pin_function (sensor, digital I/O, etc)
reset             0b11000000 (192)
ping              0b00011000 (24)  (unused)


Arduino -> RasPi Commands:
encoder_data      0b11111011       value                            (unused)
analog_data       0b11110111       value                            (unused)
digital_data      0b11101111       pin#                              value
distance_wait     0b11011110       (unused)                         (unused)
distance_data     0b11011111       data part 1                      data part 2
error             0b11111111       value
pin_ack           0b00001111 (15)
startup           0b11000001 (193)


Error Codes:
invalid_pin       0b00000001       pin#
invalid_packet    0b00000010
